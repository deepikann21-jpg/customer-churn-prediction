# 📊 Customer Churn Prediction

A Machine Learning web app to predict whether a telecom customer will churn or not.

## 🔧 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn (Random Forest Classifier)
- Streamlit for UI

## 📁 Dataset
Telecom Customer Churn Dataset containing customer demographic, account, service, and financial information.

## 📌 Features
- Gender, Senior Citizen, Partner, Dependents
- Tenure, Contract Type, Payment Method
- Monthly Charges, Total Charges
- Internet Service, Phone Service and more

## ✅ How to Run Locally
```
pip install -r requirements.txt
streamlit run app.py
```
